# OpenTrial

**Intern:** Reuben N Addison
**Project Type:** Computation Engine

## Overview
OpenTrial is a Bayesian trial design engine that generates clinical trial reports from precedent data, literature evidence, safety signals, and target biology context.

## Deliverable
- Streamlit form for trial design inputs
- Bayesian sample size and operating characteristic report
- Downloadable Markdown/PDF report with citation provenance

## Core Tools
- ClinicalTrials.gov
- PubMed E-utilities
- openFDA FAERS
- Open Targets
- WHO ICTRP
- DailyMed
- PharmGKB

## Tech Stack
Python, Streamlit, PyMC, pandas, `google-generativeai`

## Notes
The project is intended as a rigorous clinical development planning tool with automated report generation.
